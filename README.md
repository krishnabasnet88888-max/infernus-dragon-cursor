# infernus-dragon-cursor
🐉 INFERNUS — An interactive dark fantasy dragon that follows your cursor. Built with vanilla JS, SVG SMIL animations, canvas particle effects, fire breath, animated gradients, pulsing glow filters, lava background &amp; rotating rune sigil. Zero dependencies. Pure HTML/CSS/JS.
